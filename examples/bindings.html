Types of Bindings

Display Bindings 
<span data-bind="text: firstName"></span>

<input type="text" data-bind="value: firstName" />

<span data-bind="html: pageSnippet"></span>

<div data-bind="visible: showGreeting">Hi!</div>

<input data-bind="enable: inputEnabled" />

<input data-bind="disable: inputDisabled" />

<div data-bind="css: { bold: showAsBold, red: showAsRed }">Hi!</div>

<span data-bind="style: { color: textColor }">Hi!</span>

<div data-bind="attr: { id: itemId}">Hi!</div>
Flow Control Bindings 
<table>
  <tbody data-bind="foreach: tasks">
    <tr><td>Repeat!</td></tr>
  </tbody>
</table>

<div data-bind="with: childValue" >
  <span data-bind="text: childProp"></span>
</div>

<div data-bind="if: showGreeting">Hi!</div>

<div data-bind="ifnot: showGreeting">Grumble, grumble</div>
Interaction Bindings 
click binding - calls function on the ViewModel on the click event

event binding - calls a function on the ViewModel when the specified event fires
 - Can be any DOM event.  Exmaples: keypress, mouseover, mouseout
 
submit binding - typically bound to the form element. Tiggers function on submit.
 
hasfocus binding - Does the bound element have the focus?
 
checked binding - Used instead of the value binding for checkboxes and radio buttons

options binding - Supplies a dyanmic list of options to a select element

selectedOptions binding - Allows for binding to a multi-select element.
 
uniqueName binding - Adds a unique name property to the bound element, mainly used to support jQuery validation.
